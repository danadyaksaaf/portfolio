# Portfolio Website

This is a simple portfolio website that showcases the work and skills of the portfolio owner. The website features a dark mode for improved accessibility and user experience.

## Project Structure

The project consists of the following files:

- `index.html`: The main entry point of the portfolio website, containing the homepage structure and links to other pages.
- `about.html`: A page that provides information about the portfolio owner, including background, skills, and experience.
- `projects.html`: A showcase of completed projects, including descriptions, technologies used, and links to live demos or repositories.
- `contact.html`: A page that includes a contact form or information for visitors to reach out to the portfolio owner.
- `css/style.css`: The stylesheet for the website, including styles for dark mode, layout, typography, and responsive design.
- `js/script.js`: JavaScript code for handling dark mode toggling, form validation, and interactive elements.
- `assets/fonts`: A directory containing font files used in the website for custom typography.

## Features

- Dark mode toggle for better user experience.
- Responsive design for accessibility on various devices.
- Sections for about, projects, and contact information.

## Setup Instructions

1. Clone the repository or download the project files.
2. Open `index.html` in a web browser to view the portfolio website.
3. Modify the content in `about.html`, `projects.html`, and `contact.html` as needed to personalize the portfolio.

## Technologies Used

- HTML
- CSS
- JavaScript
